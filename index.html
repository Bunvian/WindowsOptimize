Các bước tối ưu hóa
1
Cập nhật Windows
Đảm bảo hệ thống được cập nhật với các bản vá lỗi và driver mới nhất để tối ưu hiệu suất và bảo mật.
Settings → Update & Security → Windows Update
Nhấn "Check for updates"
Cập nhật các bản vá, driver quan trọng
2
Cài đặt Driver
Kiểm tra và cài đặt các driver thiếu để đảm bảo phần cứng hoạt động tối ưu.
Mở Device Manager để kiểm tra driver thiếu
Tải driver từ trang web chính thức
Dell, HP, Asus, MSI, Lenovo...
3
Xóa ứng dụng rác
Loại bỏ ứng dụng không cần thiết để giải phóng dung lượng và tăng hiệu suất hệ thống.
Settings → Apps → Apps & features
Quét và xóa bloatware tự động
Xóa những app rác không sử dụng
4
Tinh chỉnh với Winutil
Sử dụng Winutil để tối ưu hóa Windows chuyên nghiệp với các tweaks nâng cao.
Win + X → PowerShell (Admin)
irm "https://christitus.com/win" | iex
5
Bật High Performance
Kích hoạt chế độ hiệu suất cao để tối ưu hóa CPU và hệ thống.
Kích hoạt Ultimate Performance Mode
powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
6
Cấu hình DNS
Thiết lập DNS Google để tăng tốc độ truy cập và ổn định kết nối.
Cấu hình WiFi/Ethernet Adapters
IPv4 DNS:
Primary: 8.8.8.8
Secondary: 8.8.4.4
IPv6 DNS:
Primary: 2001:4860:4860::8888
Secondary: 2001:4860:4860::8844
7
Tắt tính năng không cần thiết
Vô hiệu hóa các tính năng ảnh hưởng đến hiệu suất gaming và làm việc.
Tắt Game Mode và Xbox Services
Tắt Background Apps
Quản lý Startup Programs
8
Cài đặt ứng dụng cần thiết
Cài đặt các ứng dụng và runtime cần thiết cho hệ thống hoạt động ổn định.
WinRAR, 7-Zip
Visual C++ Redistributables
.NET Desktop Runtime
Microsoft Office
9
Tối ưu Visual Effects
Điều chỉnh hiệu ứng hình ảnh để tối ưu hóa hiệu suất với sự cân bằng giữa đẹp mắt và tốc độ.
Search → Adjust the appearance and performance of Windows
Adjust for best performance
